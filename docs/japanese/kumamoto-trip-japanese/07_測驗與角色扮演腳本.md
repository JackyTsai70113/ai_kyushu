# 測驗與角色扮演腳本

## A. 中文轉日文

請遮住答案，3 秒內說出日文。

1. 不好意思。
   すみません。

2. 請再說一次。
   もう一度お願いします。

3. 請慢一點。
   ゆっくりお願いします。

4. 我想去熊本城。
   熊本城に行きたいです。

5. 這班巴士到熊本機場嗎？
   このバスは熊本空港に行きますか。

6. 可以用信用卡嗎？
   クレジットカードは使えますか。

7. 有中文菜單嗎？
   中国語のメニューはありますか。

8. 你推薦什麼？
   おすすめは何ですか。

9. 請結帳。
   お会計お願いします。

10. 可以寄放行李嗎？
    荷物を預けられますか。

11. Wi-Fi 密碼是什麼？
    Wi-Fi のパスワードは何ですか。

12. 溫泉在哪裡？
    温泉はどこですか。

13. 可以幫我拍照嗎？
    写真を撮ってもらえますか。

14. 保存期限到什麼時候？
    賞味期限はいつまでですか。

15. 我迷路了。
    道に迷いました。

16. 我弄丟了手機。
    スマホをなくしました。

17. 我不舒服。
    体調が悪いです。

18. 請叫救護車。
    救急車を呼んでください。

19. 請寫下來。
    書いてもらえますか。

20. 我日文不太好。
    日本語があまり上手ではありません。

## B. 聽到這句時，該怎麼回答

1. 何名様ですか。
   一人です。二人です。

2. 店内ですか、お持ち帰りですか。
   店内でお願いします。持ち帰りでお願いします。

3. ご注文はお決まりですか。
   はい、これをお願いします。

4. お支払いは現金ですか、カードですか。
   カードでお願いします。現金でお願いします。

5. パスポートをお願いします。
   はい、お願いします。

6. 袋はご利用ですか。
   いりません。お願いします。

## C. 角色扮演 1：抵達熊本

你要完成：

1. 問巴士是否到熊本車站。
2. 問車資。
3. 問在哪裡下車。
4. 下車前確認是不是熊本車站。

可用句：

1. このバスは熊本駅に行きますか。
2. 運賃はいくらですか。
3. どこで降りますか。
4. ここは熊本駅ですか。

## D. 角色扮演 2：餐廳吃熊本拉麵

你要完成：

1. 表示一位。
2. 問有沒有中文菜單。
3. 問推薦。
4. 點熊本拉麵，不要蔥。
5. 刷卡結帳。

可用句：

1. 一人です。
2. 中国語のメニューはありますか。
3. おすすめは何ですか。
4. 熊本ラーメンを一つお願いします。ネギ抜きでお願いします。
5. カードでお願いします。

## E. 角色扮演 3：旅館入住

你要完成：

1. 說要辦理入住。
2. 說有預約。
3. 出示護照。
4. 問 Wi-Fi 密碼。
5. 問溫泉時間。

可用句：

1. チェックインをお願いします。
2. 予約しています。
3. パスポートです。
4. Wi-Fi のパスワードは何ですか。
5. 温泉の利用時間は何時から何時までですか。

## F. 角色扮演 4：購物買伴手禮

你要完成：

1. 問價格。
2. 問保存期限。
3. 問能不能帶上飛機。
4. 請包裝成禮物。
5. 說不需要袋子。

可用句：

1. これはいくらですか。
2. 賞味期限はいつまでですか。
3. 飛行機に持ち込めますか。
4. プレゼント用に包んでください。
5. 袋はいりません。

## G. 角色扮演 5：手機遺失

你要完成：

1. 說手機不見。
2. 說可能忘在車上。
3. 問遺失物中心在哪裡。
4. 請對方寫下來。

可用句：

1. スマホをなくしました。
2. 車内に忘れたかもしれません。
3. 遺失物センターはどこですか。
4. ここに書いてもらえますか。

## H. 畢業檢核

你可以準備出發的最低標準：

1. 01 的急救句可以不看文字說出 10 句。
2. 02 到 06 每份文件至少能說出 5 句。
3. 07 的 A 測驗答對 16 題以上。
4. 5 個角色扮演都能完成，不必文法完美。
5. 卡住時能馬上說：
   すみません。日本語があまり分かりません。ゆっくりお願いします。
